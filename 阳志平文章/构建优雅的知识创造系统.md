# 构建优雅的知识创造系统
[原文](https://mp.weixin.qq.com/s?__biz=MzA3MzM0MjUyMQ==&mid=2652149604&idx=1&sn=3c96ebfe992694e5c57affe9ef5ba33f&scene=19#wechat_redirect)

> 知识管理不重要，重要的是掌握生成任意领域知识体系的能力。

* 太多的人将知识管理和效率工具等同
* 形式知识可以管理，暗默知识只能在相互作用和情景中习得。
* 形式知识、暗默知识转换
  * 形式 -> 形式 -- 联结化
  * 形式 -> 暗默 -- 内在化
  * 暗默 -> 暗默 -- 共同化
  * 暗默 -> 形式 -- 表出化
